Need help with SaferTab? Find answers below or reach out to us directly.

Getting Started

  1. Install SaferTab from the Chrome Web Store.
  2. Click the SaferTab icon in your browser toolbar.
  3. Create a parent password when prompted.
  4. Add a child profile and configure blocklists, time limits, and schedules.
  5. SaferTab runs automatically in the background — no further setup needed.

Troubleshooting

Extension icon is grayed out

Make sure the extension is enabled. Go to chrome://extensions, find SaferTab, and toggle it on. If you just installed it, try refreshing the page.

Sites aren't being blocked

Check that you have an active child profile with a blocklist or the adult content filter enabled. Open SaferTab, unlock with your password, and verify the profile's settings. Rules update automatically after saving.

Forgot your parent password

Click "Forgot password?" on the lock screen. You'll need to reset the extension, which clears all profiles and settings. This is a security measure — passwords are hashed and cannot be recovered.

Time limits not working

Ensure a daily time limit is set for the active profile. SaferTab tracks browsing time in the background using Chrome alarms. If the browser was closed and reopened, the timer continues from where it left off for that day.

Browsing history not showing

History is recorded only while a child profile is active. If you just set up a profile, history will appear after the child visits their first site. History is encrypted and can only be viewed after unlocking with your parent password.

Extension not working after Chrome update

Occasionally, Chrome updates may require extensions to reload. Go to chrome://extensions and click the reload button on SaferTab. Your data and settings are preserved.

Frequently Asked Questions

Is my data sent anywhere?

No. All data stays on your device in Chrome's local storage. SaferTab has no servers and makes no network requests. See our Privacy Policy for details.

Can my child uninstall SaferTab?

They would need access to chrome://extensions to remove it. You can restrict this by setting up a supervised Chrome profile for your child through Google Family Link.

Does SaferTab work in Incognito mode?

By default, extensions are disabled in Incognito. You can enable SaferTab for Incognito windows in chrome://extensions > SaferTab > Details > "Allow in Incognito."

Can I use SaferTab on multiple computers?

Yes, but each installation is independent. Profiles and settings do not sync between devices since all data is stored locally.
