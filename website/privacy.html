Privacy Policy

Effective date: March 31, 2026

Overview

SaferTab is a Chrome extension that provides parental controls for child web safety. It is designed with a simple privacy principle: your data never leaves your device. There are no accounts, no servers, and no data collection. Everything the extension stores lives in your browser's local storage and is never transmitted anywhere.

What Data the Extension Collects

SaferTab does not collect, transmit, or share any personal data. The extension does not communicate with any external servers. There is no telemetry, no analytics, and no crash reporting built into the extension.

The extension stores the following information locally on your device only, using Chrome's built-in chrome.storage.local API:

  • Child profile names and their individual settings (blocklists, allowlists, time limits, schedules)
  • Browsing history logs for monitored profiles
  • Daily usage statistics (minutes browsed, sites visited, sites blocked)
  • Parent password hash and salt (the password itself is never stored)

How Data Is Stored and Protected

All data stored by SaferTab remains on your local device and is protected using industry-standard cryptographic methods via the Web Crypto API:

  • Browsing history is encrypted using AES-GCM (256-bit) encryption
  • Encryption keys are derived using PBKDF2 key derivation
  • The parent password is hashed with SHA-256 and a unique random salt — the plain-text password is never stored

Since no data is transmitted to external servers, there is no risk of server-side data breaches. Uninstalling the extension removes all stored data from your browser.

Extension Permissions

SaferTab requests the following Chrome permissions, each required for specific functionality:

Permission Why It's Needed
storage Store profiles, settings, browsing history, and usage data locally on your device.
tabs Read the current tab's URL to enforce blocklists, allowlists, and time limits.
webNavigation Detect page navigations to track browsing history and enforce blocking rules before pages load.
declarativeNetRequest Block requests to adult content domains and blocklisted sites at the network level.
declarativeNetRequestFeedback Provide feedback on which requests were blocked, used for the blocked-sites counter on the dashboard.
alarms Run periodic checks (e.g., updating daily usage counters) without keeping a background page open.
host_permissions: <all_urls> Required to monitor and block websites across all domains. Without this, the extension could not enforce parental controls on arbitrary sites.

Third Parties

The SaferTab extension does not integrate with, send data to, or receive data from any third-party services. There are no ads, no affiliate links, and no third-party SDKs embedded in the extension.

Website Analytics

This marketing website (safertab.com) uses Google Analytics (measurement ID: G-ZVJ25667QB) to understand general website traffic, such as page views and visitor counts. This analytics tracking applies only to this website and is completely separate from the Chrome extension. The extension itself contains no analytics or tracking code of any kind.

Children's Privacy

SaferTab is a tool designed for parents and guardians to manage their children's browsing experience. The extension is installed and configured by an adult, and all settings are protected behind a parent password.

SaferTab does not knowingly collect personal information from children. In compliance with the Children's Online Privacy Protection Act (COPPA), the extension is operated and controlled entirely by a parent or guardian. No data from any user — child or adult — is ever transmitted off the device.
